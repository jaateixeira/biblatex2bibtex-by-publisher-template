From a collection of bib files (compatible to BibLaTeX/Biber), create a single bibtex file (compatible with BibTeX) for a given Latex template provided by a publisher. 

Supported LaTeX templates: 
- 'non-strict'
- 'Wiley-NJDv5'
- 'Wiley-book'
- 'Springer-sn-jnl'
- 'Springer-LNCS'
- 'IEEE-tran'
- 'ACM-acmart'
- 'Elsevier-elsarticle'
- 'Sage-sagej'
